# integracao_n8n
Comunicaçao de software externo com webhook

Requisitos:
- n8n
- Uma planilha para armazenamento de dados
- Uma credencial para api do google sheets


Como usar:
1. Abra *workflow-exemplo.json* e cole no n8n
2. Coloque suas credenciais
1. Ative o webhook
2. Rode o codigo no terminal com: node .\integracaon8n.js
Inclui:
Workflow exemplo
Formataçao de numero pra facilitar no armazenamento dos dados
Menu no terminal
